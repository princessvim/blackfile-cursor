# blackfile-cursor

A browser-based tool for inspecting base64-encoded data visually, related to the broken cursor (part of Filian ARG).

It renders the base64 text as a character grid on a canvas. Useful for spotting patterns, possibly ASCII art.

Controls:

* row width
* font size
* character highlighting
* rotation
* mirroring
* color inversion
* text reversal
